##Project Description 
This project is a platform that helps reduce food waste by connecting surplus food providers with NGOs. 
It enables restaurants, hotels, and individuals to donate leftover food instead of discarding it. 
NGOs can easily view available food, claim it, and distribute it to those in need. 
The system promotes sustainability while supporting hunger relief in an efficient and organized way.
